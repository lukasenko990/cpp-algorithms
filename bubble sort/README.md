# Bubble sort
This algorithm takes array of strings , array of ints and size number which is the size of the arrays. The arrays are sorted ordered by numbers. If numbers are equal the order is determined by string.

# Example

# IN:
5
Adam 3
Sikorka 4
Szewc 1 
Pieta 1
Mazury 3

# OUT:
Pieta 1
Szewc 1
Adam 3
Mazury 3
Sikorka 4
